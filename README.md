Visual NUnit
============

http://github.com/jimgraham/visual-nunit

## Description

A fork of the Visual Nunit [SourceForge] repository

From the [SourceForge] page

Visual Nunit is NUnit integration for Visual Studio 2008/2010. Provides convenient view to test cases and enables debugging tests easily inside development environment. Implemented as Visual Studio Integration Package. IRC: irc.freenode.net #visualnunit 

The main project's webpage is on [BubbleCloud].

## Requirements

- Visual Studio 2008, or
- Visual Studio 2010
- to build, the Visual Studio SDK
